Centro de Terapia Respiratoria
El programa gira en torno al espacio de rehabilitación de terapia respiratoria, el cual es un programa de la universidad Visión de las Américas. para facilitar las prácticas de los estudiantes de esta misma y creando una IPS universitaria para las personas con problemas respiratorios. Está ubicado en el centro de Medellín, edificio de clínica odontológica anteriormente conocido como hotel Ambasador. Ocupando el primer y segundo nivel para este centro de terapia.
El primer nivel esta modificado para que corresponda al área inicial del centro de terapia, el cual es la valoración del estado del paciente y así saber cómo se procede a realizar una terapia personalizada. Esta zona cuenta con espacios para la revisión de prueba pulmonar, 3 consultorios con su respectivo archivo, cuarto técnico de oxígeno, y baños adaptados para personas con movilidad reducida.

2do Nivel
El segundo nivel es donde se ubica toda la parte de rehabilitación. Se cuenta con simulación de una casa donde se enseña a las personas a moverse en un entorno conocido para poder ser independientes a la hora de desplazarse. El área de rehabilitación cardiopulmonar está provista con equipo adecuado para monitorear mientras el usuario está en la terapia en curso, como lo es la telemetría. Esta zona es abierta para que se pueda aprovechar la visual ya que se cuenta con ventanales completos en su fachada.
